Interface iDeterminationQuestionData

The iDeterminationQuestionData-interface contains all data that is collected during questioning a single player a single question.

property

question: iDeterminationQuestion - the data that will be sent to the client when asking the question

property

options: iDeterminationOption[] - the four answer-options for the question containing three wrong answers and the correct one

property

correct: string - the answer-ID of the correct answer

property

questionTime?: Date - the time the question has been asked

property

explanation?: string - an explanation for the question

property

timeCorrection?: number - the current amount of time that will be subtracted from the meassured time difference between the moment the question is sent to the client and the moment the answer is received
